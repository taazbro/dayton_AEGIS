# AEGIS Web Showcase

Landing page for AEGIS Autonomous Cyber Defense System

## Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
cd aegis-web
vercel
```

## Features

- Responsive design
- Live metrics display
- Sponsor showcase
- GitHub integration
- Modern glassmorphism UI

## Tech Stack

- Static HTML/CSS
- Vercel hosting
- GitHub links
